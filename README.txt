== Description ==

### AltTextGeneratorAI.com automatically generates descriptive alt text for your images upon upload using AI to help you rank faster on Google and other search engines.

#### Rank faster on search engines with SEO optimized alt text descriptions and increase visibility using AltTextGeneratorAI.**

**Automatic:** Alt text is automatically added to every image uploaded on the site.

**Customizable:** Manually update and edit the AI generated alt text.

**Easy to set up:** Just choose a plan and start generating.

**Free credits:** Get 2 free credits upon sign up.

== Installation ==

1. **Visit** Plugins > Add New
2. **Search** for "Alt Text Generator AI"
3. **Install and Activate** Alt Text Generator AI from your Plugins page
4. **Visit** [https://alttextgeneratorai.com](https://alttextgeneratorai.com?utm_source=wp&utm_medium=dl) to sign up for a free.
5. **Link** your account by going to Dashboard and copying your API key into the Alt Text Generator AI plugin settings page.
6. **Done!** You can now upload an image and alt text will be generated automatically.
